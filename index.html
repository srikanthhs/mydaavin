  • Document lessons for next cycle
12 Management Pillars
🐄
Animal Productivity
Avg lactation 3–5 L/day. Target 35% farmer participation min. Vet extension monthly for all active villages. Link to fodder subsidy schemes.
TARGET: ≥1.8 L/enrolled animal/day
👥
Member Enrollment
Minimum 50 members per MPCS. Low-volume high-consistency societies are under-enrolled. Doorstep Aadhaar-linked onboarding reduces friction by 40%.
TARGET: 20% member growth in 15 societies by Q2
📅
Collection Consistency
NDDB target: ≥90% collection days/month (≥24 of 27). Below 80% signals operator absence, transport failure, or member dropout.
TARGET: 45 of 50 active societies ≥90% consistency
🌡️
Cold Chain Integrity
Milk must reach chilling center within 4 hours. Route optimisation (Route Planner tab) reduces transit. BMC installation recommended for high-volume routes.
TARGET: Zero warm-milk rejections at depots
💰
Transparent Pricing
Two-axis fat+SNF payment. Display price board at every society. WhatsApp/SMS payment slip prevents disputes and is top reason for reactivating inactive societies.
TARGET: 100% digital payment slips by June 2026
📱
Digital Farmer Linkage
WhatsApp groups per society for daily alerts. Missed-collection auto-SMS to farmer. Monthly performance report to society chairman. ePashuhaat market linkage.
TARGET: 80% farmer mobile enrollment by Sep 2026
🔄
Route Efficiency
Optimised routes reduce fuel cost 15–30%. Kollidam 8 active societies can be covered in <4h. Add new societies to existing routes where proximity allows.
TARGET: <4h route time per block per shift
🧪
Quality Assurance
Fat ≥3.0%, SNF ≥8.5%. Adulteration check at chilling center. Seasonal fat dip of 0.3–0.5% in summer. Breed improvement (Jersey/HF crossing) for better yield.
TARGET: Zero adulteration; fat avg ≥3.2%
🌾
Seasonal Planning
Delta region dip during Samba harvest (Oct–Dec) when agricultural labor peaks. Pre-season farmer engagement essential. Identify year-round vs seasonal lactators.
TARGET: <10% seasonal variance in monthly collection
🏆
Society Grading & Incentives
Grade A societies: ₹0.20/L extra bonus. Grade D: 3-month improvement plan or merger. Public display of society grade at panchayat builds healthy competition.
TARGET: 30 Grade A societies by March 2027

TARGET: 15 new societies formed FY 2026-27

Step 2 — Deploy Apps Script Web App
1. In your Sheet: Extensions → Apps Script
2. Replace Code.gs with the script below
3. Click Deploy → New deployment → Web App
4. Execute as: Me · Who has access: Anyone
5. Click Deploy → Copy the Web App URL
6. Paste URL in "Write to Sheets" tab above
// MYL Milk Intel — Google Apps Script Webhook // Paste this in Extensions → Apps Script → Code.gs // Then Deploy as Web App (Execute as: Me, Access: Anyone) const SHEET_NAME = 'DailyCollection'; const ESC_SHEET = 'Escalations'; function doPost(e) { try { const payload = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); // Handle daily collection data if (payload.type === 'daily') { const sh = ss.getSheetByName(SHEET_NAME) || ss.insertSheet(SHEET_NAME); if (sh.getLastRow() === 0) { sh.appendRow(['Date','Society','Morning_L','Evening_L', 'Total_L','Avg7Day','PctOfAvg','Status','Notes']); sh.getRange(1,1,1,9) .setBackground('#1e8e3e') .setFontColor('#ffffff') .setFontWeight('bold'); } // Remove existing rows for same date const date = payload.date; const data = sh.getDataRange().getValues(); for (let i = data.length - 1; i >= 1; i--) { if (data[i][0] == date) sh.deleteRow(i + 1); } // Append new rows payload.rows.forEach(r => sh.appendRow([ r.date, r.society, r.morning, r.evening, r.total, r.avg7day, r.pctOfAvg, r.status, r.notes ])); return okResponse(`Wrote ${payload.rows.length} rows for ${date}`); } // Handle escalations if (payload.type === 'escalations') { const sh = ss.getSheetByName(ESC_SHEET) || ss.insertSheet(ESC_SHEET); sh.clearContents(); sh.appendRow(['Date','Society','Issue','Action', 'Officer','Level','Status']); payload.rows.forEach(r => sh.appendRow([ r.date,r.soc,r.issue,r.action, r.officer,r.level,r.status ])); return okResponse(`Wrote ${payload.rows.length} escalations`); } return errResponse('Unknown payload type'); } catch(err) { return errResponse(err.message); } } function okResponse(msg) { return ContentService .createTextOutput(JSON.stringify({ok:true,msg})) .setMimeType(ContentService.MimeType.JSON); } function errResponse(msg) { return ContentService .createTextOutput(JSON.stringify({ok:false,msg})) .setMimeType(ContentService.MimeType.JSON); } // GET handler — return sheet data as JSON (optional) function doGet(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sh = ss.getSheetByName(SHEET_NAME); if (!sh) return okResponse('No data yet'); const rows = sh.getDataRange().getValues(); return okResponse(JSON.stringify(rows)); }
⚠️ Common Issues & Fixes
CORS error on fetch: Your sheet isn't published. Go to File → Share → Publish to web, choose CSV, and ensure "Anyone with the link" is selected.
Push returns 401: Re-deploy the Apps Script as a new version, making sure "Who has access" is set to "Anyone" (not just org).
Data not appearing: Check column headers exactly match the expected format. Society names must match exactly (case-sensitive).
Fetch blocked by browser: Use a CORS proxy or switch to the Apps Script webhook for both read and write.
